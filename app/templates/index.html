The random_filipino_api is an open-source API designed to provide developers with a convenient way to generate random Filipino user information. It can be useful for testing, mock data generation, and more.

The random_filipino_api was developed using a combination of technologies for both the backend and frontend:

  • Backend: SQLite, Python (Flask)
  • Frontend: HTML, Bootstrap

These technologies were used to create a robust and user-friendly API for generating random Filipino user data. If you have any questions or need further details about the implementation, feel free to reach out.

üßë Random User

Endpoint: /api/random-user
HTTP Method: GET
Description: Returns random information about a Filipino user, including their name, address, date of birth, email, phone number, identification, and bank details.
Parameters Description
size (integer, optional)
 Number of users to retrieve. Default is 1. Maximum limit: 30.
gender (string, optional)
 Filter users by gender. Accepted values are "male" or "female".
